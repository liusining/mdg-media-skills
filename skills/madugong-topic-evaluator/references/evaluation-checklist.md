# 选题判断自检清单

最终输出前逐项检查。

## 流程自检

- [ ] 已经读取 `references/topic-selection-method.md`。
- [ ] 已经读取 `references/search-method.md`。
- [ ] 已完成网络搜索。
- [ ] 已覆盖背景、最新变化、矛盾/反证三个方向。
- [ ] 已向用户确认共同信息场、最新变化、行动建议。
- [ ] 用户已经确认，或明确授权按当前理解继续。

## 三要素自检

- [ ] 共同信息场不是专业黑话，普通观众能理解入口。
- [ ] 最新变化有明确时间、事件、数据或政策支撑。
- [ ] 行动建议存在，或明确说明为什么暂不需要行动建议。
- [ ] 没有把旧问题包装成新变化。

## 八方向自检

- [ ] 只选择主匹配和必要的次匹配。
- [ ] 没有强行把八个方向全部套上。
- [ ] 每个匹配方向都有材料路径。
- [ ] 已指出看似匹配但不适合采用的方向。

## 否定选题自检

- [ ] 已检查自己是否愿意分享。
- [ ] 已检查是否为完美故事、鸡汤故事或猎奇故事。
- [ ] 已检查是否为纯反驳式选题。
- [ ] 已生成带 `[Tn]` 的压缩叙事。
- [ ] 已统计转折点数量。
- [ ] 已判断结构切换次数。
- [ ] 已判断传播性价比。

## 结论自检

- [ ] 结论属于“值得做 / 值得做但需要重构 / 暂缓 / 不建议做”四类之一。
- [ ] 已给出最大风险。
- [ ] 已给出改造建议。
- [ ] 已给出更好的选题表述或标题方向。
- [ ] 所有事实性判断都有引用。
