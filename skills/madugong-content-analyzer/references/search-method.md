# 搜索方法

来源：马督工自媒体课（搜索部分）

## 1. 利用学术资源

在开动通用搜索引擎之前，先用学术搜索工具查相关论文（找最热和最新的，哪怕只看摘要和结论，也能为后续搜索指明方向）。

## 2. 如何找到搜索关键词

核心原理（香农信息论）：**关键词不怕错，就怕太普通。** 信息量 = 消除的不确定性。大概率出现的通用词信息量低，搜索结果必然差；小概率的特殊词信息量高，能快速指向有效结果。

具体方法：
- **选择不可预测的词**：在原始资料中找最独特的字词，而非标题或通用名称。类比"外号比名字更能精确表达特征"。
- **利用可预测的偏差**：错别字、俗称、非官方说法本身就是关联信号，偏差方向可预测时反而能帮助定位。
- **组合普通词制造稀缺性**：单个常见词无效，但不常见的组合同样满足高信息量条件。
- **执着翻页**：至少看50页结果再考虑换关键词，有效内容藏在干扰信息之间。
- **灵活迭代**：每轮搜索结果都要即时学习，调整下一轮关键词。

## 3. 找茬游戏：从重复内容的差异中提取信息

搜索结果大量重复时不要放弃——比较这些相似内容之间的微小差异，差异的方向往往不是随机的，而是和每个信息源的利益直接相关。

规律：**本钱越大的信息源，表述越谨慎。** 小媒体/自媒体用确定语气说满话，大媒体/官方媒体在正文中留退路——标题强势但正文含糊，堆砌材料但不说因果。这种带方向性的偏差本身就是线索，说明核心事实可能经不起验证。同一事件中不同企业的评价差异、前后两届政府工作报告对同一数据的用词变化，都能用同样的方法提取潜在的内容方向。

## 4. 将搜索过程组合成叙事逻辑

搜索应尽量在第二轮或第三轮得到最广信息，然后回顾自己的搜索和学习过程，自然形成内容结构。确定结论后，倒转因果关系——从深层信息往回追溯到通用信息，找出一条捷径，给观众提供不超过两层转折的逻辑链。搜索方向越多越好，但展示给观众的路径要经过简化。
