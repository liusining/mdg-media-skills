# 马督工内容分析 Checklist

## 1. 话题/选题拆分标准

不要按标题数量机械拆题。只有当一段内容拥有独立中心问题、独立因果链、独立转折或独立建议，并且拆出来也能自成一篇文章时，才算独立选题。

常见误判：

- “现象—原因—解决方案”是同一选题的三段，不是三个选题。
- “经济原因—性别原因—政策建议”通常是同一选题的递进，不是三个选题。
- 主持人换问题、章节换标题、引用换来源，不一定表示换选题。

## 2. A/B/C/D 判定口径

- A 叙述：事实、数据、引用、现场、历史材料。
- B 逻辑：因果解释、利益关系、系统机制、成本收益分析。
- C 点缀：增强趣味或现场感，可删但会损失可读性。
- D 转折：改变读者预期或改变论证方向。D 节点必须服务核心媒体价值。

## 3. 转折点计数口径

只统计“删掉就无法复述主线”的转折。

不计入：

- 时间推进：后来、然后、接着。
- 举例切换：比如、另外、再看。
- 普通让步：虽然辛苦但仍然辛苦。
- 主持人为了推进节目而提出的新问题。

计入：

- 表层判断被推翻。
- 责任主体被重新定位。
- 问题从个案变成结构。
- 问题从批判对象转向解决方案。
- 解决方案被反转为另一个更可执行方案。

## 4. Mermaid 检查

- 使用合法 Mermaid flowchart 语法。
- 主线必须连通。
- D 节点用菱形更清晰。
- C 节点作为旁支，不要喧宾夺主。
- 并列节点最后应汇入一个 B 或 D 节点。

## 5. 选题成立分析

必须先判断“共同信息场 + 最新变化 + 行动建议”，再匹配八个选题方向。八方向不是打标签，而是解释为什么这个话题值得做。

优先寻找：

- 普通生活中的异常数据。
- 热点背后的结构性原因。
- 群体内部矛盾和成本分配。
- 可让观众用自身经验参与讨论的入口。
- 数据、历史、合订本能够揭示的新变化。
